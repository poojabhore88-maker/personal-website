# Deploy Instructions

## Push to GitHub
```bash
git push -u origin main
```

## Deploy to Vercel

### Option 1: Vercel Dashboard (Easiest)
1. Go to [vercel.com](https://vercel.com)
2. Sign up/login with GitHub
3. Click "New Project"
4. Import `personal-website` repository
5. Click "Deploy"

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel --prod
```

## Your Website Will Be Live At:
https://personal-website-[your-username].vercel.app
